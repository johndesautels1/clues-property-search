## Usage

### Basic Workflow
1. **Select Properties**: Click property cards in the sidebar to select up to 3 properties
2. **Choose Analysis Mode**: Select comparison mode from dropdown (Detailed, Quick, Portfolio)
3. **Run Analysis**: Click "Run Analysis" to generate insights
4. **Explore Charts**: Interact with various visualization types
5. **Export Reports**: Generate PDF reports (simulated)

### Chart Types
- **Radar Charts**: Multi-attribute comparison
- **Vector Momentum**: Price trend analysis
- **Topography Maps**: Value density visualization
- **Time Series**: Historical price tracking
- **Matrix Analysis**: Property-to-property comparisons

## Technical Details

### Dependencies
- **D3.js v7**: Data visualization library
- **Luxon**: Date/time manipulation
- **Font Awesome 6**: Icon library
- **Google Fonts (Inter)**: Typography
- **Chroma.js**: Color manipulation

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Data Structure
The application uses a comprehensive property data model including:
- Basic property information (price, size, bedrooms)
- Market data (assessed value, market value, estimates)
- Neighborhood metrics (school ratings, walk scores, crime scores)
- Historical price data
- Comparative market analysis

## Customization

### Adding New Properties
Edit `js/data.js` to add new property entries following the existing structure.

### Styling
- Professional theme: `css/professional.css`
- Fighter jet theme: `css/fighter-jet.css`

### Chart Configuration
Modify `js/charts.js` to adjust visualization parameters and add new chart types.
